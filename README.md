# Automatic Medicine Dispenser

**Role:** Hardware Designer & Programmer  
**Team:** Team of 3
**Tech:** Proteus, 8086 Assembly, LCD, Keypad  

## Project Overview
This project simulates an automatic medicine dispenser that can manage 3 separate dispensers for 3 users simultaneously. The system integrates a keypad, LCD, sound alarm, and an inventory monitor to provide a clear and interactive interface for medication scheduling.

## Features
- Multiple dispensers support
- User input via keypad
- User passwords
- LCD displays current user and schedule
- Alarm for doses
- Stock monitor LED and refill
- Assembly code optimized for 8086 microprocessor
